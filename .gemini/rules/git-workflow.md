Jika pengguna meminta untuk push atau commit kode, selalu gunakan format Conventional Commits (sesuaikan dengan 'git log -5' untuk pola commit terakhir).

Urutan eksekusi untuk Frontend (FE / direktori sdn-2-kalimati):
1. git add .
2. git commit -m "..."
3. git push

Urutan eksekusi untuk Backend (BE / direktori api-sdn-2-kalimati):
1. npm run build
2. git add .
3. git commit -m "..."
4. git push
