## Tech stack
- Next.js 16 + React 19, Turbopack (dev & build)
- Tailwind CSS v4 via `@tailwindcss/postcss` (PostCSS plugin, NOT v3 config)
- Flowbite React (wraps `next.config.ts` via `flowbite-react/plugin/nextjs`)
- TypeScript strict, `@/*` alias → project root
- Dark mode via `class` strategy, `localStorage` key `sdn2-theme`
- Font: Poppins (Google Fonts)

## Auth
- Token stored in `sessionStorage` + mirrored to cookie `user_session` (max-age 86400)
- Middleware (`app/middleware.ts`) protects `/dashboard` and redirects logged-in users away from `/login`
- API base: `https://api-sdn-2-kalimati.vercel.app/api`
- API client (`lib/api.ts`) reads token from `sessionStorage`, sends as `Bearer` header

## Project structure
```
app/             — Next.js App Router pages & layouts
  components/    — Shared UI (BackButton, Pagination, LoadingModal, etc.)
  data/          — Static data (class list)
  types/         — RegistrationForm interface
  reactbits/     — TextType component
lib/             — API client, constants
services/        — AuthService, RegistrationService (static class methods)
types/           — User interface (server-side model)
```

## Architecture notes
- Registration form uses `RegistrationService.create()` → POST `/api/registration`
- Dashboard fetches via `RegistrationService.getAll()` → GET `/api/registration`
- Validation toggle: PATCH `/api/registration/:id` with `{ status: "validated"|"unvalidated" }`
- Dashboard pagination: 5 items per page
- Graduation check (`/kelulusan`) uses local static data, not API
- `.flowbite-react/` is auto-generated by Flowbite; `class-list.json` sourced in CSS
- No `.env` files committed (`.env*` in `.gitignore`)
