- TypeScript strict, `@/*` alias → project root
- Dark mode via `class` strategy, `localStorage` key `sdn2-theme`
- Font: Poppins (Google Fonts)

## Auth
- Token stored in `sessionStorage` + mirrored to cookie `user_session` (max-age 86400)
- Middleware (`app/middleware.ts`) protects `/dashboard` and redirects logged-in users away from `/login`
- API base: `https://api-sdn-2-kalimati.vercel.app/api`
- API client (`lib/api.ts`) reads token from `sessionStorage`, sends as `Bearer` header

## Project structure

```
app/             — Next.js App Router pages & layouts
  components/    — Shared UI (BackButton, Pagination, LoadingModal, etc.)
  data/          — Static data (class list)
  types/         — [deprecated] Prefer `types/` at root

lib/             — API client, constants, CSV utils, format helpers
services/        — AuthService, RegistrationService (static class methods)
types/           — Shared types: user.ts, registration.ts, attendance.ts
```

## Architecture notes
- Registration form uses `RegistrationService.create()` → POST `/api/registration`
- Dashboard fetches via `RegistrationService.getAll()` → GET `/api/registration`
- Validation toggle: PATCH `/api/registration/:id` with `{ status: "validated"|"unvalidated" }`
- Dashboard pagination: 5 items per page
- Graduation check (`/kelulusan`) uses local static data, not API
- `.flowbite-react/` is auto-generated by Flowbite; `class-list.json` sourced in CSS
- No `.env` files committed (`.env*` in `.gitignore`)

## Holiday System

### Attendance integration
- `hooks/usePresensi.ts` fetches holidays on mount via `HolidayService.getAll()`
- `DateDayPicker` accepts `blockedDates` prop — disables holiday dates in calendar (grayed + line-through)
- When selected date is a holiday: amber banner shown "Hari Libur — Tidak bisa mencatat presensi pada tanggal ini"
- Save button disabled when `isHoliday`
- Backend also rejects holiday saves with "Tidak bisa mencatat presensi di hari libur!"

| File | Tugas |
|------|-------|
| `services/holiday.service.ts` | API client: `getAll()`, `add()`, `remove()` |

| `hooks/usePresensi.ts` | Edited: fetch holidays + `isHoliday` + `holidayList` + `refreshHolidays` |
| `app/(admin)/presensi-murid/page.tsx` | Edited: blockedDates + holiday banner + disabled save + **manage holiday modal (admin only)** |

### Manage holiday UI (admin/kepala only)
- Tombol "Atur Libur" di samping DateDayPicker pada halaman presensi
- Modal: tambah libur (input date + deskripsi) + daftar libur dengan tombol hapus
